# 专业分析引擎 (Major Analyzer)

## 职责

基于就业数据、行业趋势、AI替代风险评估专业选择。将张雪峰的「就业倒推法」系统化为可量化的分析框架。

## 专业分析六维度

### 维度1: 就业率

**数据来源**：教育部公布的各专业就业率、麦可思研究院《中国大学生就业报告》、智联招聘/猎聘年度报告。

```json
{
  "major": "计算机科学与技术",
  "employment_rate": 0.94,
  "related_employment_rate": 0.78,
  "avg_salary_5yr": 18000,
  "salary_growth_5yr": 0.15,
  "unemployment_risk": "low"
}
```

### 维度2: 薪资中位数（不是平均数）

**核心原则**：不看前3%进大厂的，不看后5%找不到工作的，看中间50%的毕业生5年后的收入。

| 专业门类 | 起薪中位数(税前/月) | 5年后中位数 | 10年后中位数 | 薪资成长性 |
|---------|-------------------|------------|-------------|----------|
| 计算机类 | 8,000-12,000 | 15,000-25,000 | 25,000-50,000 | 高 |
| 电子信息类 | 7,000-10,000 | 12,000-20,000 | 20,000-40,000 | 高 |
| 临床医学类 | 5,000-8,000 | 10,000-18,000 | 15,000-35,000 | 中高（需规培） |
| 金融学类 | 6,000-10,000 | 10,000-20,000 | 15,000-40,000 | 高方差（看平台） |
| 机械类 | 5,000-7,000 | 8,000-13,000 | 12,000-20,000 | 中 |
| 土木类 | 5,000-7,000 | 7,000-12,000 | 10,000-18,000 | 低（行业下行） |
| 法学类 | 4,000-7,000 | 8,000-15,000 | 10,000-30,000 | 中（需过法考） |
| 新闻传播类 | 4,000-6,000 | 6,000-10,000 | 8,000-15,000 | 低 |
| 师范类（公办教师） | 5,000-8,000 | 8,000-12,000 | 10,000-15,000 | 稳但低 |
| 生物科学类 | 4,000-6,000 | 6,000-10,000 | 8,000-15,000 | 低（需读博） |

### 维度3: AI替代风险评估

| 风险等级 | 标志 | 代表专业 | 策略 |
|---------|------|---------|------|
| 极高 | 标准化、重复性高、不需要深度人际交互 | 基础翻译、基础会计、部分法律文书 | **避坑** |
| 高 | 可被AI辅助大幅提升效率 | 初级程序员(CRUD)、基础设计、数据分析入门 | 必须提升到「会用AI」的层次 |
| 中 | AI增强但不可替代 | 中高级开发、建筑设计、金融分析 | 学专业+AI工具 |
| 低 | 需要深度人际交互/物理操作 | 临床医学、牙医、护理、电气工程现场 | 稳定，AI是工具 |
| 极低 | 高度依赖人判断/信任/伦理 | 法官、心理咨询、教育 | 护城河深 |

### 维度4: 不可替代性评分

```
不可替代性 = 技术壁垒 × 行业门槛 × 人际不可替代性

- 技术壁垒（0-10）：需要多少专门训练才能胜任？
  - 计算机/Al: 7-8 | 机械: 6-7 | 中文/历史: 2-3
- 行业门槛（0-10）：入行需要什么证/学历/背景？
  - 临床医学: 9（医师资格证+规培） | 法学: 7（法考） | 销售: 1
- 人际不可替代性（0-10）：AI能替代多少人际交互？
  - 心理咨询: 9 | 护士: 8 | 程序员: 3
```

### 维度5: 天坑专业识别

张雪峰经典「生化环材四大天坑」+ 扩展版：

| 专业 | 天坑等级 | 判别标准 |
|------|---------|---------|
| 生物工程/生物技术 | ★★★★★ | 本科就业率极低，必须读博 |
| 环境工程 | ★★★★ | 行业规模小，岗位集中在政府/国企 |
| 化学/化工 | ★★★★ | 工作环境差，薪资增长慢 |
| 材料科学（基础方向） | ★★★★ | 本科难就业，读博才有出路 |
| 土木工程（2024+） | ★★★★ | 房地产下行，新增 |
| 英语/小语种（2025+） | ★★★ | AI翻译冲击，新增 |
| 工商管理（普通院校） | ★★★ | 万金油但没硬技能 |
| 行政管理 | ★★★ | 考公是唯一出路 |
| 旅游管理 | ★★★ | 行业受经济周期冲击大 |

### 维度6: 考公/考编适配度

| 适配等级 | 专业 | 可报岗位数 | 竞争比 |
|---------|------|----------|--------|
| 极佳 | 法学、汉语言文学、会计学、计算机 | 很多 | 中等 |
| 良好 | 经济学、金融学、统计学、公共管理 | 较多 | 较高 |
| 一般 | 英语、新闻学、社会学 | 一般 | 高 |
| 较差 | 生物、化学、环境 | 少 | 很高 |
| 极差 | 艺术类、体育类 | 极少 | 极高 |

## 「就业倒推法」执行流程

```
Input: 考生分数 + 省份 + 意向专业方向
Output: 专业推荐列表 + 风险警告

Step 1: 根据意向专业，查询该专业类别的就业率+薪资中位数
Step 2: 查询该专业在各省/各层次院校的就业差异
Step 3: 计算AI替代风险
Step 4: 识别是否为「天坑专业」
Step 5: 给出明确建议：
    - 这个专业能不能学？
    - 如果能，什么层次的学校值得去？
    - 如果不能，推荐什么替代方向？
Step 6: 如果分数不够好学校，建议是否走「先专业后学校」策略
```

## 特色大学/专业解析模板

当用户询问具体院校/专业时，按以下模板输出：

```
【{学校名} - {专业名}】

毕业生中位数去向：
- 5年后薪资中位数：{金额}
- 主要去向行业前3：
- 读研比例：{百分比}
- 出国比例：{百分比}

这个专业在业内的真实口碑：
- 正面：{3条}
- 负面：{3条}
- 特别注意：{1条关键信息}

张雪峰式评价：
"我跟你说，这个学校的这个专业，{一句话结论}。你看它{x数据}，{x对比}。你要是{x条件}，那可以试试。你要是{x条件}，趁早换个方向。"

同类院校对比：
| 学校 | 该专业录取位次 | 该专业就业率 | 该专业薪资中位数 | 推荐指数 |
|------|------------|------------|----------------|---------|
| A | {位次} | {就业率} | {薪资} | ★★★★☆ |
| B | {位次} | {就业率} | {薪资} | ★★★☆☆ |
```
