# 信息采集引导 (Intake Guide)

## 职责

在生成任何志愿填报报告前，引导用户提供充分的信息。信息不足时，绝不跳过直接给建议——这正是张雪峰「灵魂追问法」的工程化版本。

## 采集流程

按层级递进，每层确认后再进入下一层。用户信息不完整的层级，用反问方式主动索取。

---

## 第一层：考试基础信息（必填）

### 1.1 省份与考试类型
- **省份**：？（如浙江、河南、广东）
- **高考模式**：老高考文理 / 3+3 / 3+1+2？
- **科类**：理科/物理类 / 文科/历史类 / 不分文理？

### 1.2 成绩信息
- **高考分数**（或预估分）：？
- **全省位次**（如果已知）：？
- **批次线**：一本线/特殊类型线/一段线过了多少分？

### 1.3 选科（新高考省份必填）
- **选科组合**：？（如 物理+化学+生物 / 历史+政治+地理）
- **各科大致成绩或等第**：？

### 1.4 学考等第（浙江/上海等三位一体/综合评价省份特别重要）
- **学考等第**：A几个？B几个？
- 这直接影响三位一体/综合评价的报名资格和入围概率

---

## 第二层：个人画像（强烈建议）

### 2.1 性格与兴趣
- **MBTI或性格自述**：偏内向/外向？喜欢和人打交道还是和机器打交道？
- **兴趣爱好**：平时喜欢做什么？有什么特长？
- **学科偏好**：最喜欢和最讨厌的科目？
- **擅长的能力**：逻辑推理 / 语言表达 / 动手操作 / 创意设计 / 组织协调？

### 2.2 职业倾向
- **想做的工作类型**：
  - [ ] 技术/研发（写代码、搞工程、做实验）
  - [ ] 与人打交道（销售、咨询、教育、医疗）
  - [ ] 创意/内容（设计、写作、媒体、艺术）
  - [ ] 稳定/体制内（公务员、事业单位、国企）
  - [ ] 商业/管理（创业、管理、金融、投资）
  - [ ] 还没想清楚
- **绝对不能接受的工作**：？（如：不想996/不接受异地/不想出差）

---

## 第三层：偏好约束

### 3.1 地域偏好
- **目标省份/城市**：？（如：就想在浙江/能接受去外省/想去一线城市）
- **不能接受的地域**：？（如：不去东北/不去西北/不去太偏的地方）
- **省内 vs 省外**：更倾向于留本省还是出省？各省离家多远能接受？

### 3.2 院校偏好
- **学校层次追求**：
  - [ ] 非985/211不上
  - [ ] 一本就行
  - [ ] 有学上就行
  - [ ] 无所谓，主要看专业
- **学校类型偏好**：综合性大学 / 理工科院校 / 财经类 / 师范类 / 医药类？
- **校园环境偏好**：大城市 / 大学城 / 无所谓？

### 3.3 专业倾向
- **感兴趣的专业方向**（可多选）：
  - [ ] 计算机/人工智能/软件
  - [ ] 电子信息/通信/微电子
  - [ ] 机械/电气/自动化/机器人
  - [ ] 临床医学/口腔医学/药学
  - [ ] 数学/物理/化学等基础科学
  - [ ] 金融/经济/会计/管理
  - [ ] 法学
  - [ ] 师范/教育
  - [ ] 中文/新闻/外语
  - [ ] 建筑/土木/设计
  - [ ] 生物/环境/材料
  - [ ] 还没想法
- **绝对不能接受的专业**：？（如：不学医/不学师范/不进工厂）

### 3.4 学费预算
- **年学费承受范围**：
  - [ ] 5000以内（公办普通专业）
  - [ ] 5000-20000（中外合作/软件工程等高收费专业）
  - [ ] 20000以上（中外合作办学/民办院校）
  - [ ] 不限

---

## 第四层：家庭背景（张雪峰核心维度）

> 「有矿和没矿，策略完全不同」——张雪峰

### 4.1 经济条件
- **家庭年收入大致范围**：？
- **是否需要申请助学贷款**：？
- **能否支持读研/读博（多供3-7年）**：？
- **能否支持出国留学**：？

### 4.2 家庭资源
- **父母/直系亲属所在行业**：？
  - 这个行业内是否有可以帮上忙的人脉？
  - 家族是否有企业/生意可以接手？
- **家人在哪些城市有生活基础**：？（有亲戚在的城市，毕业后有落脚点）

### 4.3 家庭期望
- **父母对孩子专业的期望**：？
- **家庭对孩子就业的期望**：希望稳定（考公/国企）还是追求高薪？
- **家族有无传承压力**：需要回乡接班/照顾父母/其他约束？

---

## 第五层：特殊条件加分项

### 5.1 政策性加分
- **是否有高考加分**：（少数民族/烈士子女/归侨/退役士兵等）加多少分？
- **加分适用**：是全国通用还是仅省属高校？

### 5.2 专项计划资格
- [ ] **国家专项**：贫困地区户籍+3年学籍？
- [ ] **高校专项**：边远贫困民族地区农村户籍？
- [ ] **地方专项**：本省农村户籍？
- [ ] **公费师范生**：是否愿意签6年服务期（含2年农村）？
- [ ] **定向医学生**：是否愿意去基层医疗机构服务？
- [ ] **定向农林**：是否愿意接受？

### 5.3 特殊才能
- **竞赛获奖**：五大学科竞赛省一及以上？其他全国性竞赛？
- **强基计划**：是否对数学/物理/化学/生物/基础医学等基础学科有兴趣，且成绩突出？
- **综合评价**：是否已报名某校的综评（如南科大/上纽大/昆山杜克/国科大等）？
- **艺术/体育特长**：是否有高水平艺术团/运动队资格？

### 5.4 身体条件
- **视力**：是否近视超过4.8？色盲/色弱？（影响军警校/飞行/航海/化工等专业）
- **身高/体重**：是否符合军警校标准？
- **其他**：有无重大疾病史/传染病史？

## 信息采集进度指示器

在采集过程中，向用户展示信息完整度：

```
信息采集进度：
✅ 第一层：考试基础 [○ 完整 / ● 待补充：位次、学考等第]
✅ 第二层：个人画像 [● 完整]
⚠️ 第三层：偏好约束 [● 待补充：地域偏好、学费预算]
✅ 第四层：家庭背景 [○ 完整]
⬜ 第五层：特殊加分 [待采集]
━━━━━━━━━━━━━━━━━━━━
完整度：60% → 建议至少完成前三层再生成报告
```

## 最低信息门槛

**生成正式志愿报告的最低信息要求**：
- 第一层：省份 + 分数 + 科类（必填）
- 第三层：至少1个偏好维度（地域/专业/院校）
- 第四层：至少家庭经济条件（有矿/没矿）

**信息不足时**：
- 不生成完整报告，改为给出「方向性分析」+ 追问缺失信息
- 明确标注："以下分析基于不完整信息，仅供参考，建议补充XX信息后重新分析"

## 浙江特殊采集项

针对浙江考生，额外采集：

1. **学考等第详情**：因为三位一体报名条件通常要求"几A几B"
2. **是否参加过三位一体报名**：哪些学校？
3. **选考科目（7选3）的分数/位次**：浙江是3+3，选考科目成绩影响专业选择
4. **一段线/二段线**：浙江分两段录取，80个平行志愿
5. **是否考虑高职提前招生**：
