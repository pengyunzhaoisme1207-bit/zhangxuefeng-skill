# 冲稳保风险评估引擎 (Risk Assessor)

## 职责

基于考生位次和目标院校历年录取位次，计算冲/稳/保三级风险，输出阶梯志愿方案。

## 核心算法

### 位次差计算

```
位次差 = 考生位次 - 目标院校专业组最低录取位次
```

- 正值 → 考生位次高于录取位次（更安全）
- 负值 → 考生位次低于录取位次（有风险）

### 冲稳保阈值

| 批次 | 冲 (Reach) | 稳 (Match) | 保 (Safety) |
|------|-----------|-----------|------------|
| 老高考一本 | 位次差 -5%~0% | 位次差 0%~+10% | 位次差 +10%~+25% |
| 新高考本科批 | 位次差 -8%~0% | 位次差 0%~+12% | 位次差 +12%~+30% |
| 新高考（专业+院校） | 位次差 -10%~0% | 位次差 0%~+15% | 位次差 +15%~+35% |

> 新高考「专业+院校」模式下，热门专业的实际位次波动更大，冲的阈值应放宽。

### 阶段定义

| 阶段 | 英文 | 定义 | 位次差范围 | 录取概率 |
|------|------|------|-----------|---------|
| 冲刺 | Reach | 考生位次略低于录取位次，有捡漏可能 | -5%~0% | 10-30% |
| 稳健 | Match | 考生位次与录取位次匹配或略高 | 0%~+10% | 50-80% |
| 保底 | Safety | 考生位次显著高于录取位次 | +10%~+25% | 90%+ |
| 极限冲 | Aggressive | 今年可能爆冷或新增招生 | -10%~-5% | <10% |
| 绝对保 | Guarantee | 绝对有学上的底线 | +25%+ | 99%+ |

## 志愿梯度设计

### 老高考省份（院校平行志愿）

- 共 6 个院校志愿（各省不同，4-10 个）
- 推荐分配：冲 2个 + 稳 2个 + 保 2个

### 新高考（院校专业组）

- 共 45 个志愿（各省不同，30-96 个）
- 推荐分配：冲 10-15个 + 稳 15-20个 + 保 10-15个

### 新高考（专业+院校，如浙江/山东）

- 共 80-96 个志愿
- 推荐分配：冲 15-20个 + 稳 30-40个 + 保 20-30个

## 风险评估因子

除了位次差，以下因子会影响实际录取概率：

### 加分因子（位次更安全）
- 今年该校该专业扩招（+5%~+10%安全边际）
- 新增校区/中外合作项目首次招生（+15%安全边际）
- 该校去年爆冷分数异常低（存在均值回归风险，不作为冲的依据）

### 减分因子（位次更危险）
- 今年缩招（-5%~-10%安全边际）
- 新高考第一年（数据参考价值降低，波动±15%）
- 热门专业（计算机/AI/临床/口腔）位次年涨（-5%~-10%安全边际）
- 网红院校/专业（媒体曝光增多→报名热度上升→位次上涨）

## 防撞车分析

2026年高考可能出现的热点方向（结合当前趋势预测）：

| 方向 | 风险等级 | 原因 |
|------|---------|------|
| 人工智能/AI专业 | 高位次上涨 | 行业热点+媒体炒作，分数可能被推高 |
| 新能源/储能科学 | 中高位次上涨 | 政策风口，新增专业增多 |
| 集成电路/微电子 | 高位次上涨 | 卡脖子领域+国家重视 |
| 临床医学（5+3一体化） | 持续高位 | 稳定的刚需专业 |
| 师范类（公费/定向） | 持续热门 | 编制吸引力在经济下行期更强 |
| 土木工程/建筑类 | 低位次下跌 | 房地产下行，吸引力下降 |
| 小语种/部分文科 | 低位次下跌 | AI翻译冲击+就业面窄 |

## 志愿表生成算法

```
function generate_plan(score, province, category, preferences):
    1. 查询一分一段表 → 位次
    2. 查询所有院校在该省的录取位次 + 招生计划变化
    3. 过滤：排除不在目标地域/学费超预算的院校
    4. 排序：按位次差从小到大
    5. 分类：
       冲：位次差 -5%~0%     → 取 top 2-15个
       稳：位次差 0%~+10%    → 取 top 2-30个  
       保：位次差 +10%~+25%  → 取 top 2-20个
    6. 去重：同一院校不同专业组去重
    7. 检查：确保保底院校至少有 2 个在 +15% 安全线内
    8. 输出志愿表 + 每个志愿的录取概率区间
```
