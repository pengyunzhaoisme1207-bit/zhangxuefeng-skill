# 政策合规检查引擎 (Policy Checker)

## 职责

根据考生省份、年份、科类、选科组合，检查：
1. 是否有资格报考目标院校/专业
2. 是否满足特殊招生条件
3. 是否踩到政策红线

## 一、高考模式识别

### 全国高考模式分布（2025-2026）

| 模式 | 省份 | 志愿填报方式 | 志愿数量 |
|------|------|------------|---------|
| 3+3 | 北京、天津、上海、浙江、山东、海南 | 专业+院校（浙鲁）/ 院校专业组 | 80-96 / 24-60 |
| 3+1+2 | 河北、辽宁、江苏、福建、湖北、湖南、广东、重庆、吉林、黑龙江、安徽、江西、广西、贵州、甘肃 | 院校专业组 | 40-45 |
| 传统文理 | 河南、四川、山西、陕西、内蒙古、宁夏、青海、新疆、西藏、云南 | 院校平行志愿 | 6-12 |
| 新高考首年(2025) | 吉林、黑龙江、安徽、江西、广西、贵州、甘肃 | 院校专业组 | 40-45 |
| 新高考首年(2026) | 河南？(待确认) | 待定 | 待定 |

> 每年高考前应更新本表。数据来源：各省教育考试院官网。

### 选科要求检查

新高考省份的选科组合必须满足院校专业组的选科要求。

| 专业类 | 常见选科要求 |
|--------|------------|
| 计算机类/电子信息类 | 物理（几乎全部）|
| 临床医学 | 物理+化学 或 化学+生物 |
| 口腔医学 | 物理+化学 或 化学+生物 |
| 化学/化工类 | 化学 |
| 生物科学类 | 物理+化学 或 化学+生物 |
| 机械/电气/土木/建筑类 | 物理 |
| 数学/统计类 | 物理 |
| 金融/经济类 | 不限 或 物理（少数）|
| 法学/新闻/中文/外语 | 不限 |
| 师范类 | 不限 或 对应学科 |

**选科不匹配 = 直接排除，不可填报。**

## 二、特殊招生类型

### 强基计划
- **目标**：选拔基础学科拔尖人才
- **专业**：数学、物理、化学、生物、力学、基础医学、历史、哲学、古文字学
- **院校**：39所原985高校
- **时间**：3-4月报名，高考后校测
- **注意**：入围看高考成绩（85%）+校测（15%），有一定降分空间

### 综合评价招生
- **目标**：综合素质优秀的学生
- **典型院校**：南方科技大学、上海纽约大学、昆山杜克大学、中国科学院大学（部分省份）
- **注意**：需要提前报名，有面试/校测环节

### 国家专项计划
- **对象**：贫困地区农村户籍考生
- **院校**：中央部属高校+地方重点高校
- **优势**：降分幅度5-20分
- **条件**：需具有实施区域连续3年户籍+学籍

### 高校专项计划
- **对象**：边远/贫困/民族地区县及以下农村学生
- **院校**：95所重点高校（含所有985/211）
- **优势**：降分幅度可达30-60分

### 地方专项计划
- **对象**：各省农村户籍学生
- **院校**：各省省属重点高校

### 公费师范生
- **待遇**：免学费+生活费补助+毕业包分配
- **限制**：需服务6年（含农村2年）
- **注意**：违约需赔偿，记录诚信档案

### 军警校/消防
- **额外要求**：政审、体检、体能测试、面试
- **时间**：提前批，高考后单独测试

### 艺考/体育
- **统考**：各省统一组织的专业考试
- **校考**：独立设置的艺术院校单独考试
- **文化课要求**：逐年提高

## 三、批次与录取规则

### 批次划分

| 批次 | 含义 | 典型院校 |
|------|------|---------|
| 本科提前批 | 军警校/公费师范/定向医学生/飞行员等 | 军校、警校、公费师范、定向医学生 |
| 特殊类型批 | 强基/综评/高水平运动队/艺术团 | - |
| 本科批（一本） | 普通本科一批 / 新高考本科批 | 985/211/双一流及部分省重点 |
| 本科批（二本） | 普通本科二批（部分省合并） | 普通本科院校 |
| 专科批 | 高职高专 | 职业技术学院 |

### 平行志愿录取规则

```
分数优先：所有考生按分数（位次）排序，高分先投档
遵循志愿：对每个考生，按志愿顺序逐个检索
一轮投档：每人每批次只有一次投档机会
```

**关键影响**：
- 一旦投档到A校但被退档（如不服从调剂），本批次后续志愿全部作废 → 只能等征集志愿或下一批次
- 「服从专业调剂」是防退档最后防线

## 四、政策红线检查清单

回答志愿填报问题前，必须过此清单：

- [ ] 考生选科组合满足目标专业组要求？（新高考省份）
- [ ] 考生户籍/学籍满足特殊招生（专项计划）条件？
- [ ] 考生身体条件满足军警校/飞行员/航海类专业要求？
- [ ] 目标专业对外语语种/单科成绩有要求？
- [ ] 中外合作办学项目学费是否在预算内？
- [ ] 是否提醒了「不服从调剂」的退档风险？
- [ ] 是否检查了志愿梯度的合理性（冲+稳+保都有）？

## 五、各省特殊政策速查

### 北京
- 实行「3+3」模式，不分文理
- 本科普通批30个志愿
- 高水平艺术团/运动队有单独通道

### 上海
- 实行「3+3」模式
- 本科普通批24个院校专业组
- 综合评价批次独立

### 河南
- 目前仍为传统文理分科
- 2026年可能启动新高考（待确认）
- 一本6个院校志愿，二本9个
- 高考人数全国最多（2025年约136万）

### 广东
- 实行「3+1+2」模式
- 本科批45个院校专业组
- 物理类/历史类分别填报

### 四川
- 目前仍为传统文理分科
- 一本9个院校志愿，二本9个
- 即将启动新高考改革

### 浙江
- 实行「3+3」模式
- 一段线志愿80个（专业+院校）
- 是全国最早的试点省份
