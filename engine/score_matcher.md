# 分数匹配引擎 (Score Matcher)

## 职责

将考生分数+省份转化为全省位次，再映射到可报考的院校/专业组。

## 输入

| 参数 | 说明 | 示例 |
|------|------|------|
| province | 省份 | 河南 |
| score | 高考分数 | 612 |
| category | 科类 | 理科/物理类 / 文科/历史类 |
| year | 考试年份 | 2025 |
| batch | 批次 | 本科一批 / 本科批 |

## 工作流

### Step 1: 分→位转换

通过一分一段表（`data/yifenyiduan/{province}_{year}.json`）将分数转换为全省位次。

```
位次 = 一分一段表中该分数对应的累计人数
```

**2025 一分一段数据结构**：
```json
{
  "province": "henan",
  "year": 2025,
  "total_examinees": 1360000,
  "segments": [
    {"score": 750, "count": 5, "cumulative": 5},
    {"score": 749, "count": 3, "cumulative": 8}
  ]
}
```

### Step 2: 位→校匹配

对于每个候选院校/专业组，查找其近3年在目标省份的最低录取位次。

**录取数据数据结构**（`data/scores/{province}_{year}.json`）：
```json
{
  "province": "henan",
  "year": 2025,
  "batch": "本科一批",
  "schools": [
    {
      "code": "10001",
      "name": "北京大学",
      "category": "理科",
      "min_score": 698,
      "min_rank": 95,
      "avg_score": 701,
      "avg_rank": 72,
      "enrollment": 45,
      "majors": [
        {"name": "计算机类", "min_rank": 65, "enrollment": 12},
        {"name": "数学类", "min_rank": 78, "enrollment": 10}
      ]
    }
  ]
}
```

### Step 3: 位次映射表（快速查）

| 省 | 理科一本线附近位次（2025参考） | 文科一本线附近位次 |
|----|--------------------------|-------------------|
| 河南 | ~120,000 | ~25,000 |
| 四川 | ~80,000 | ~18,000 |
| 广东 | ~85,000（物理） | ~22,000（历史） |
| 北京 | ~27,000 | ~5,000 |
| 湖北 | ~65,000 | ~14,000 |
| 山东 | ~130,000（一段线） | ~100,000（一段线） |
| 江苏 | ~90,000（物理） | ~30,000（历史） |
| 湖南 | ~80,000（物理） | ~25,000（历史） |
| 浙江 | ~90,000（一段线） | ~90,000（一段线） |

> **数据更新原则**：每年高考出分后，一分一段表官方公布，应立即更新 `data/yifenyiduan/`。使用 `tools/data_updater.md` 中的流程。

### Step 4: 新高考省份适配

新高考省份（3+1+2 或 3+3）使用「院校专业组」或「专业+院校」模式。

- **3+1+2**（河北/辽宁/江苏/福建/湖北/湖南/广东/重庆/吉林/黑龙江/安徽/江西/广西/贵州/甘肃）：物理类/历史类分别对应原理科/文科
- **3+3**（北京/天津/上海/浙江/山东/海南）：不分文理，但有选科要求
- **老高考**（河南/四川/山西/陕西/内蒙古/宁夏/青海/新疆/西藏/云南）：理科/文科

**选科匹配检查**（`data/policies/subject_requirements.json`）：
- 计算机/电子信息类通常要求物理
- 临床医学通常要求物理+化学，或化学+生物
- 金融/经济学通常不限选科
- 法学/新闻/管理类通常不限选科

## 搜索关键词模板

当数据库中没有目标省/年份数据时，使用以下搜索策略：

```
省级考试院官网 site:xx省教育考试院 2025年普通高校招生 录取投档线
一分一段表 site:xx省教育考试院 2025年
{年份}年{省份}高考{院校}录取分数线 位次
{省份}高考志愿填报 2025 投档线位次表
```

## 注意事项

- 位次比分数更稳定（每年试题难度不同，但位次相对稳定）
- 招生计划的增减会影响位次波动（计划增加→位次可能下降）
- 新高考省份第一年数据波动大，需参考更多年份
- 热门专业（计算机/AI/临床医学）的位次可能逐年上升
